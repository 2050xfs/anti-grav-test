# Tailwind Color Configuration

## Color Choices

- **Primary:** `indigo` — Used for buttons, links, key accents, active navigation states
- **Secondary:** `emerald` — Used for growth metrics, positive indicators, success states
- **Neutral:** `slate` — Used for backgrounds, text, borders, inactive states

## Usage Examples

Primary button: `bg-indigo-600 hover:bg-indigo-700 text-white`
Secondary badge: `bg-emerald-100 text-emerald-800 dark:bg-emerald-950 dark:text-emerald-100`
Neutral text: `text-slate-600 dark:text-slate-400`
Border: `border-slate-200 dark:border-slate-800`
Background: `bg-slate-50 dark:bg-slate-950`

## Dark Mode

All components include dark mode variants using the `dark:` prefix. Ensure your root element supports dark mode toggling with a `dark` class.
